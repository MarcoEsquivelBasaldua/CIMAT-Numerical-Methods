Code description:

Inputs:
- M_MEDIUM.txt
    File contenant a matrix
    the first row in M_MEDIUM.txt has two numbers referring to number of rows and columns in the matrix
    the matrix is then presented by rows and columns.
    
Outputs:
- None

Compiler instructions:
- By positioning in the folder where Max_Index.c and M_MEDIUM.txt are stored, use gcc compiler: 
    gcc Max_Index.c
- Run: 
    ./a.out M_MEDIUM.txt
- Results:
    Maximum absolute value in the matrix is located
    its location is stored
    Results after execution are displayed in the terminal window.
