Code name: Boole.c
Author: Marco Antonio Esquivel Basaldua
Date: 03/11/19

Code description:
User selects a function present in the start menu to get its integral value
by Boole method for the given interval.
Results are presented in the terminal window.

The results are displayed in the terminal screen
    
Inputs:
- None
    
Outputs:
- None

Compiler instructions:
- By positioning in the folder where Boole.c is stored, use gcc compiler: 
    gcc Boole.c -lm
- Run: 
    ./a.out
- Results:
    approximation by Boole rule for the integral given the function chosen
    is displayed in the terminal window
