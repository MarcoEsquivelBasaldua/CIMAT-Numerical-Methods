Code name: Simson1_3.c
Author: Marco Antonio Esquivel Basaldua
Date: 03/11/19

Code description:
User selects a function present in the start menu to get its integral value
by Simson 1/3 method for the given interval.
Results are presented in the terminal window.

The results are displayed in the terminal screen
    
Inputs:
- None
    
Outputs:
- None

Compiler instructions:
- By positioning in the folder where Simson1_3.c is stored, use gcc compiler: 
    gcc Simson1_3.c -lm
- Run: 
    ./a.out
- Results:
    approximation by Simson 1/3 rule for the integral given the function chosen
    is displayed in the terminal window
