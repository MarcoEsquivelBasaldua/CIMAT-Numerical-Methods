Code description:
This Code realizes the matrix vector multiplication
and stores the result in one vector using parallelization.
The procedure is done using 1, 2, 3 and 4 threads
    
Inputs:
- None

    
Outputs:
- "Results.txt" The time results for each number of thread
- "Matrix times Vector.png" graph time vs number of threads

Compiler instructions:
- By positioning in the folder where matXvect.c is stored, use the Makefile placed in this folder, just write "make" press enter in the terminal

- Results:
    Results after execution are displayed in the terminal window
