Code name: vectSum.c
Author: Marco Antonio Esquivel Basaldua
Date: 05/10/19

Code description:
This Code realizes the sum of two vectors and stores the result in another vector
using parallelization.
The procedure is done using 1, 2, 3 and 4 threads
    
Inputs:
- None

    
Outputs:
- "Results.txt" The time results for each number of thread
- "Sum of Vectors.png" graph time vs number of threads

Compiler instructions:
- By positioning in the folder where vectSum.c is stored, use the Makefile placed in this folder, just write "make" press enter in the terminal

- Results:
    Results after execution are displayed in the terminal window
