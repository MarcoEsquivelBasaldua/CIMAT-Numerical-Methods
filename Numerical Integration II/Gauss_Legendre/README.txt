Code name: GaussLegendre.c
Author: Marco Antonio Esquivel Basaldua
Date: 05/11/19

Code description:
User selects a function present in the start menu to get its integral value
by Gauss-Legendre method for the given interval using n.
The value of n must be provided by the user, it can go from 0 to 4 
Results are presented in the terminal window.

The results are displayed in the terminal screen
    
Inputs:
- None
    
Outputs:
- None

Compiler instructions:
- By positioning in the folder where GaussLegendre.c is stored, use gcc compiler: 
    gcc GaussLegendre.c -lm
- Run: 
    ./a.out
- Results:
    Approximation by Gauss-Legendre rule using n for the integral given the function chosen.
    Results are displayed in the terminal window
